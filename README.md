## Run Locally

Clone the project

```bash
  git clone https://github.com/plsankar/colorhunter
```

Go to the project directory

```bash
  cd colorhunter
```

Install dependencies

```bash
  npm install
```

Run development build

```bash
  npm run dev
```

or run the production build

```bash
  npm run build
```
